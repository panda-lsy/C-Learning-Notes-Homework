# 分支结构

在 1.2-2 的程序中，已经完成了四则运算程序界面的输出，需要**分支结构**来选择需要的选项

```mermaid
flowchart LR
A([开始])-->B([输入Choice])
B-->C{判断Choice}
C-->D(加法)
C-->E(减法)
C-->F(乘法)
C-->G(除法)
C-->H(退出)
```

此时需要`if`语句来进行选择

```c
if (条件){
}
```

## 编写一个分段函数

编写输出如下分段函数值的程序，要求整数$x$的值从键盘输入

$$
y=\left\{  
     \begin{array}{lr}  
     x+1(x\leq0)  \\  
     1(0<x\leq3) \\  
     x (x>3)&  
     \end{array}  
\right.
$$

对于程序，我们可以绘制一个流程图

```mermaid
flowchart LR
A([开始])-->B[定义整型变量x,y]-->C[输入x]-->D{x<=0}
D--T-->E[y=x+1]
E-->F{"x<=3 && x>0"}
D--F-->F
F--T-->G[y=1]
G-->H{x>3}
F--F-->H
H--T-->I[y=x]
I-->J[输出y]
H--F-->J
J-->K(结束)

```
