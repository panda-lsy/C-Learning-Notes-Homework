# 循环

## `while`循环

`while`语句的一般形式

```c
while (表达式){
	循环体
}
```

### 例题

见2.1-1.c

## `for`循环

`for`语句的一般形式

```c
for (表达式1;表达式2;表达式3){
	循环体
}
```

`for`语句的执行过程

```mermaid
graph LR
F["开始"]-->A["处理表达式1"]-->B{"表达式2"}--非0-->C["执行循环体"]-->D["处理表达式3"]-->B
B--0-->E["结束"]
```
